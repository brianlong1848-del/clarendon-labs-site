# Clarendon Labs Website

The marketing site for [Clarendon Labs](https://clarendon.dev), an independent app studio based in Chicago.

## Getting started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS
- Deployed on Vercel

## Pages

- `/` — Home
- `/privacy` — Privacy Policy
- `/support` — Support
- `/terms` — Terms of Service
